## 🎯 项目概述

**项目名称**: Lkyjj — AI 全栈产品+研发 个人作品集网站  
**版本**: v2.0 (已实现国际化)  
**在线地址**: https://portfolio-lkyjj.vercel.app

## 📋 项目定位

打造集「成果呈现、技术演示、AI交互、国际化展示」于一体的个人品牌网站，核心特色：

1. **数据驱动展示**: 用具体数据证明能力（如"幻觉率32%→<5%""对话准确率92%"）
2. **技术能力可视化**: 通过雷达图、技能矩阵、项目卡片等可视化展示技术栈
3. **国际化支持**: 完整的中英文双语切换，覆盖所有页面和组件
4. **全链路能力**: 展示从需求洞察→方案设计→技术落地→数据复盘的完整能力

## 🔧 技术架构

### 前端技术栈
```
- Next.js 15 (App Router)
- React 19
- TypeScript
- TailwindCSS
- Framer Motion (动效)
- Recharts (数据可视化)
- React Icons
```

### 后端 & 部署
```
- Node.js (API Routes)
- Vercel (Serverless部署)
- GitHub API (项目数据同步)
- 本地JSON (数据存储)
```

### 核心功能实现
```typescript
// 国际化系统
lib/utils/i18n.ts - 翻译字典、语言切换

// API接口
src/app/api/
  ├── github-repos/ - GitHub项目拉取
  ├── projects/ - 实习项目数据
  └── chat/ - AI对话接口（预留）

// 数据存储
data/
  ├── projects.json - 实习项目
  ├── skills.json - 技能数据
  ├── timeline.json - 时间轴数据
  ├── knowledge-base.json - AI知识库
  └── backup-repos.json - GitHub备份
```

## 📊 页面功能详解

### 1. 关于我页面（/about）

**展示内容**:
- 个人头像与基本信息（姓名、职位、年龄、学校）
- 个人宣言（专注LLM应用落地，用产品思维驱动技术价值）
- 4大核心能力模块：AI技术、产品能力、全栈开发、工具链
- 完整经历时间轴（4段实习+2段教育）
- 联系方式（电话、邮箱、GitHub）

**国际化支持**:
- ✅ 姓名：刘康宇 / Liu Kangyu
- ✅ 职位：AI产品经理 + 全栈研发工程师 / AI Product Manager + Full-stack Engineer
- ✅ 时间轴：所有公司名、职位、成就双语
- ✅ 联系方式标签：电话/Phone、邮箱/Email

### 2. 项目展示页面（/projects）

**核心功能**:
- GitHub API自动拉取开源项目（带降级方案）

- 分类筛选：全部/产品落地类/独立开发类/AI专项类
- 项目卡片展示：名称、描述、技术栈、成果数据
- Skeleton骨架屏加载

**数据来源**:

**可视化展示**:
- 雷达图：AI技术、产品能力、前端开发、后端开发
- 列表视图：详细技能列表+熟练度进度条

**每个案例包含**:

**预留接口**: `/api/chat` (需配置AI API Key)

### 6. 联系我页面（/contact）

**两大模块**:
1. 联系信息卡片：电话、邮箱、GitHub（支持一键复制）
