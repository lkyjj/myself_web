# Portfolio LKYJJ - 个人作品集网站

## 项目概述
这是一个展示AI产品经理+全栈研发复合能力的个人作品集网站，包含7个核心模块：

- 个人简介 (About)
- 项目展示 (Projects) 
- 案例复盘 (Case Study)
- 技能矩阵 (Skills)
- AI智能交互 (AI Intro Bot) - 开发中
- 技术演示 (Playground) - 待开发
- 联系方式 (Contact) - 待开发

## 技术栈
- **前端**: Next.js 15 (App Router), React 19, TypeScript
- **样式**: TailwindCSS, Framer Motion
- **数据可视化**: Recharts
- **AI集成**: LangChain, OpenAI
- **部署**: Vercel

## 本地开发
```bash
npm install
npm run dev
```

## 部署
项目已配置Vercel自动部署，推送到main分支即可自动部署。

## 性能优化
- 启用了静态页面生成(SSG)
- 图片和资源缓存优化
- CDN加速配置
- 安全头信息配置
